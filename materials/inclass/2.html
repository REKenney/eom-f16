Tasks

GitHub —Setup Student Repository

  1. Create an account on GitHub
  2. Fork my example student repository by clicking this link: github.com/andrewringler/eom-f16-student then clicking the Fork button in the top right hand corner.
  3. Download, install and run the GitHub Desktop App. (should already be installed on school computers).
  4. Log in to your GitHub account from the GitHub Desktop App. If you have just installed it, after running it will prompt you for your username and password. If you are running from the school computer you probably have to login by clicking in the menu bar GitHub Desktop — Preferences… — Accounts
  5. From the menubar click File — Reload Repositories to make sure you are in-sync with GitHub cloud
  6. From the GitHub Desktop App, click the + (plus) button in the top left corner, then Clone, you should see eom-f16-student repository listed (if it is not listed quit and restart GitHub desktop). Then click Clone eom-f16-student in the bottom right-hand corner. Choose a location on your harddrive to save this repository locally.

GitHub —Add Homework #1 to GitHub

  1. In Finder copy your homework assignment #1 sketch folder into your local eom-f16-student folder under a sub-directory called assignment_1. You may delete my example homework :).
  2. Switch back to GitHub Desktop App, you should see “1(or more) Uncommitted Changes.” Click on Uncommitted Changes, then type a Message under Summary, click Commit to master, then Sync. You should see you changes published at github.com.

Checkout Class Examples

  1. Fork the Elements of Media website repository on GitHub.
  2. Clone locally from the GitHub Desktop App.
  3. You should now be able to browse to, and run the in class examples. They are in the materials/examples sub-folder.
